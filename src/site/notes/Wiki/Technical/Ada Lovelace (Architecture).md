[[Organizations/Nvidia\|Nvidia]]
is a ==high-performance GPU architecture released in 2022 (GeForce RTX 40-series), focusing on advanced real-time ray tracing, AI-driven graphics (DLSS 3), and superior efficiency==. Built on TSMC's 4N process, it features 3rd-gen RT cores, 4th-gen Tensor cores, and Shader Execution Reordering (SER) for up to 2x-3x faster rendering

named after [[People/Ada Lovelace\|Ada Lovelace]]
